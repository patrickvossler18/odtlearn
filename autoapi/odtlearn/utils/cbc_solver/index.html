class odtlearn.utils.cbc_solver.CBCSolver(verbose)[source]#

Bases: odtlearn.utils.solver.Solver

The CBC solver interface. This class contains functions for interacting with the solver for setting up, optimizing, and getting optimal values of a model. This class interacts with a slightly modified version of the SolverCbc class from the python-mip package.
